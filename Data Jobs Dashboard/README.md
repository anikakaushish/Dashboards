Overview -->

Created a dashboard to analyze trends in data-related job postings, salaries, and hiring demand across various job roles.

Key Insights -->

 * Compared median yearly and hourly salaries
 * Identified most in-demand data job roles
 * Tracked job posting trends over time
 * Analyzed salary distribution by job title

Features -->

 * Salary Trend Analysis
 * Job Count Visualization
 * Role-wise Salary Comparison
 * Interactive Filtering
 * Data Career Insights
