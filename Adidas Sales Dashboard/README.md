Overview -->
Built a business intelligence dashboard to monitor Adidas sales performance across regions, products, retailers, and states.

Key Insights -->

 * Evaluated total sales, operating profit, and units sold
 * Analyzed regional and state-wise sales performance
 * Identified top-selling products and retailers
 * Monitored monthly sales trends
   
Features -->

 * Geographic Sales Mapping
 * Retailer Performance Analysis
 * Product-wise Sales Analysis
 * Dynamic Date Filtering
 * Interactive Visualizations
