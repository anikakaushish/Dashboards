Overview --> 

Designed an HR analytics dashboard to analyze employee attrition, workforce demographics, and job satisfaction metrics.

Key Insights -->

 * Tracked employee attrition and retention rates
 * Analyzed attrition by department, age group, and education field
 * Evaluated job satisfaction across job roles
 * Monitored workforce distribution and active employees

Features -->

 * Employee Attrition Analysis
 * Department-wise Insights
 * Age Group & Gender Analysis
 * Job Satisfaction Matrix
 * Interactive HR KPIs
